Advanced call now button, website visitors on a mobile device can call your business with a single click.

== Description == 
This plugin adds a customizable call bar/call now button to the bottom of the website, visitors on a mobile device can call your business with a single click.  

== Installation ==
1. Navigate to your WordPress dashboard and go to Plugins > Add New > and search for Advanced Call Now Button. 
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Advanced Call Now Button to configure the plugin
4. Click enable and enter your phone number, message.
5. Click Save Changes and you're done!
